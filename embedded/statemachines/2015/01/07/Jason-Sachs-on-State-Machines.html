Jason Sachs on State Machines

07 Jan 2015

One of my favorite writers on all things electronics and embedded systems, Jason Sachs, has written a great article about State Machines.

I have always found it a bit odd that State Machines aren't more popular with "regular" programmers. At EI just about every project Dad does involves a State Machine or two. I learnt first hand how effective State Machines were many years ago when Dad bailed me out on a job I had worked on. The horrible nested if statement at the heart of my program was replaced by a simple (and more importantly reliable) state machine at the last minute in order to be able to finalise the job.

Anyway, go read Jason's article. It is an easy read on a useful subject.
